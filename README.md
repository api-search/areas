# areas.apis.io — API Evangelist Areas

[**areas.apis.io**](https://apis.io/areas/) organises the [APIs.io](https://apis.io) network by subject area. Each page is a curated collection of the specifications, tools, and documentation behind a topic — schema design, data modeling, cloud, cybersecurity standards, and more — and cross-links the API providers on the network working in that area.

Areas are the topical knowledge base of the network. They are distinct from **providers** (who ships an API), **industries** (who buys an API), and **categories** (what an API does).

**Live site:** [https://apis.io/areas/](https://apis.io/areas/)

## What's in this repo

- **`_areas/`** — one Markdown file per area, generated from the `kind: topic` repos in the [api-evangelist](https://github.com/api-evangelist) organization. Front matter carries the curated resource list, tags, and tag-matched related providers.
- **`index.html`, `_layouts/`, `_includes/`, `assets/`** — the browse and detail pages
- **`_config.yml`** — Jekyll site configuration

## How it's built

This collection is regenerated by [`network/scripts/build_areas.py`](https://github.com/api-search/network/blob/main/scripts/build_areas.py) (invoked from `build.py`), which scans every `kind: topic` repo under the API Evangelist source, renders its curated resources, and cross-links network providers by tag overlap. Those same topic repos are **excluded from the provider catalog** so they no longer appear as bogus providers on providers.apis.io. The site itself is a Jekyll build deployed via the apis.io CloudFront origin.

## The APIs.io network

| Site | Repo | What it indexes |
|---|---|---|
| [apis.io](https://apis.io) | [network](https://github.com/api-search/network) | Search hub, blog, build pipeline |
| [providers.apis.io](https://apis.io/providers/) | [providers](https://github.com/api-search/providers) | API provider profiles |
| [apis.apis.io](https://apis.io/apis/) | [apis](https://github.com/api-search/apis) | Individual API records |
| [industries.apis.io](https://apis.io/industries/) | [industries](https://github.com/api-search/industries) | Providers by industry vertical |
| [regions.apis.io](https://apis.io/regions/) | [regions](https://github.com/api-search/regions) | Providers by sales region |
| [areas.apis.io](https://apis.io/areas/) | [areas](https://github.com/api-search/areas) | Curated subject areas (topic repos) |
| [tags.apis.io](https://apis.io/tags/) | [tags](https://github.com/api-search/tags) | API tag index |

## Related projects

- [api-evangelist](https://github.com/api-evangelist) — upstream source repos this network is built from
